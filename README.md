# University Lost and Found System

A web-based system designed to help university students and staff report, search, and recover lost items efficiently.

## Overview
This project provides a centralized platform for managing lost and found items within a university environment. It replaces manual processes with a secure and organized digital solution.

## Features
- User registration and login system
- Report lost and found items
- Search and filter items
- Claim and verification system using secret details
- Internal messaging between users
- Notification system

## Technologies Used
- Python (Flask)
- MySQL Database
- HTML, CSS, JavaScript

## System Architecture
The system follows a multi-tier client-server architecture:
- Frontend: Web interface
- Backend: Flask application
- Database: MySQL
